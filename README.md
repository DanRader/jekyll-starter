# jekyll-starter
A Ready-to-go starter pack to kickoff every Jekyll projects with the basics and a few helpful tools to help along the way.

## Guiding Principles
* Well Documented
* Easy to Stand up 
* Easy to build upon
* Modular Reusable Components with minimal styles
* Capture common Jekyll logic

## Tech & Tools
* Jekyll
* Siteleaf Ready
* Sass
* Bass CSS
* Grunt
